Features

REST-API

The server provides a fully documented REST-API. You can write your own client by just using the HTTP REST-API.

Statistics

Take a look how many transactions or users your instance of strichliste had

SQLite Database

The data is stored in a SQLite database which can be easily backed up and queried

Web based

Because strichliste-web is web based, you can use it on your smartphone, tablet or on your desktop computer
